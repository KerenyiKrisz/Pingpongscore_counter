# Ping-Pong Scoreboard

A simple dashboard for keeping track of ping-pong scores between friends. Built as a fun project to easily monitor wins and losses without hassle.

## About

This is a lightweight web-based scoreboard that tracks games between two players: Krisz and Georg. Just open the page, increment or decrement scores, and you're done. It works on any device with a browser.

## Features

- Simple and intuitive interface
- Buttons to increase or decrease each player's score
- Reset button to start a new game
- Works on desktop and mobile devices
- Player profile pictures displayed
- No setup required, just share the link

## Live Demo

Open the scoreboard here: https://kerenyikrisz.github.io/Pingpongscore_counter/

## How to Use

Open the link in your browser. Use the plus button to add a point and the minus button to subtract. Hit reset to start fresh. Share the link with your friend so you can both access the same scoreboard.

## Built With

HTML5, CSS3, Vanilla JavaScript, and GitHub Pages.
